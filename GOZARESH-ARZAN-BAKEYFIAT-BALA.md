# 💰 گزارش تحقیق بازار — بهترین کیفیت با کمترین قیمت (۳ ژوئیه ۲۰۲۶)

> نتیجه جستجوی عمیق روی کل وب + مقایسه ده‌ها سرویس API برای عکس، ویدیو، و انتشار خودکار.

## 🎯 خلاصه اجرایی (پیشنهاد نهایی من)

| بخش | سرویس پیشنهادی | قیمت واقعی |
|---|---|---|
| 🖼 **عکس با چهره ثابت** | **Fal.ai → مدل `PuLID-FLUX`** یا **Runware.ai** | **$0.0006 تا $0.035 هر عکس** (بسته به مدل) |
| 🎬 **ویدیو** | **Fal.ai/Replicate → `Wan 2.6`** یا **`Seedance Fast`** | **$0.02 تا $0.07 هر ثانیه** (~$0.15-0.35 هر ویدیوی ۵ ثانیه) |
| 📤 **انتشار خودکار** | **Zernio** (که از قبل توی کدتون پیاده‌سازی شده!) | **رایگان تا ۲ اکانت، بعد $۶/اکانت در ماه** |
| 🗣 **صدا (اختیاری)** | **edge-tts** (از قبل توی کدتون هست) | **۱۰۰٪ رایگان، همیشه** |

**هزینه واقعی ماهانه برای یک اینفلوئنسر با ۳ پست در روز (عکس) + ۱ ویدیوی کوتاه در روز:**
- عکس: ۹۰ عکس/ماه × ~$0.01 = **~$1** (با Runware/FLUX Schnell)
- ویدیو: ۳۰ ویدیو/ماه × ~$0.15 = **~$4.5**
- انتشار: **$0 تا $6** (اگه فقط ۲ پلتفرم بخواید رایگانه)
- **جمع کل: تقریباً $6 تا $12 در ماه** — به‌جای گیر کردن کامل در سرویس‌های رایگان محدود.

## 🖼 بخش ۱: عکس با چهره ثابت (Face Consistency)

### مشکل فعلی که باید حل بشه
پروژه شما نیاز به **image-to-image با عکس مرجع** داره (نه فقط توصیف متنی). گزینه‌هایی که واقعاً عکس مرجع رو می‌گیرن:

| مدل | پلتفرم ارائه‌دهنده | قیمت هر عکس | نکته |
|---|---|---|---|
| **PuLID-FLUX** | **Fal.ai** | **$0.035** | بهترین کیفیت face-lock رایگان/ارزان که پیدا کردم |
| **FLUX.1 Schnell** | **Runware.ai** | **$0.0006 - $0.0019** | ارزان‌ترین گزینه در کل بازار (بدون image-to-image قوی) |
| **Gemini `gemini-2.5-flash-image` (Nano Banana)** | **Google AI (پولی)** | **$0.039** (یا $0.0195 با Batch API) | همون مدلی که الان دارید استفاده می‌کنید؛ اگه billing فعال کنید همینی که هست کار می‌کنه، فقط دیگه رایگان نیست |
| **InstantID / PuLID (SDXL)** | **Replicate** | **~$0.01-0.02** | دومین گزینه ارزان با face-lock واقعی |
| **Nano Banana Pro** | **Google/Fal** | **$0.067-0.134** | بالاترین کیفیت، ولی گرون‌تر |

**🏆 پیشنهاد من:** **Fal.ai** با مدل **PuLID-FLUX** ($0.035/عکس) چون:
- کد شما همین الان `_hf_pulid_flux_image` رو داره که دقیقاً از همین مدل استفاده می‌کنه؛ فقط باید نسخه Fal.ai (پایدارتر و بدون صف انتظار) رو اضافه کنیم
- ثبت‌نام بدون کارت (طبق تحقیق قبلی، ~$1 اعتبار رایگان اولیه هم می‌ده)
- برای شروع فقط باید کارت بانکی بین‌المللی (مجازی هم قبول می‌کنن، مثل Payoneer/Wise) اضافه کنید تا از اعتبار رایگان که تموم شد ادامه بدید

**جایگزین کاملاً رایگان-ولی-محدود که همین الان توی کدتونه:** Gemini با billing غیرفعال (کوتای رایگان روزی ~۵۰۰ درخواست بوده ولی طبق لاگ شما الان به صفر رسیده — این یعنی یا محدودیت منطقه‌ای‌تون هست یا حساب جدیدتون هنوز کامل فعال نشده).

## 🎬 بخش ۲: ویدیو

### گزینه‌های واقعی (نه mock)

| مدل | قیمت هر ویدیوی ۵ ثانیه | چهره ثابت؟ |
|---|---|---|
| **Seedance 2.0 Fast** (ByteDance، از طریق Fal/Replicate) | **~$0.11** | تا ۱۲ عکس مرجع می‌پذیره → عالی برای الینا |
| **Wan 2.6** (Alibaba) | **~$0.35** | image-to-video، عکس مرجع می‌پذیره |
| **Kling 3.0** | **~$0.63** | کیفیت سینمایی بالا |
| **Hailuo 2.3 (MiniMax)** | **~$0.20-0.40** | مناسب شبکه‌های اجتماعی |

**🏆 پیشنهاد من:** **Seedance 2.0 Fast از Fal.ai** — ارزان‌ترین با پشتیبانی از تا ۱۲ عکس مرجع برای حفظ ثبات چهره الینا در طول ویدیو، دقیقاً چیزی که لازم داریم.

**نکته پیاده‌سازی:** پروژه شما همین الان `video_generator.py` (DirectorAgent) و `faceless_studio.py` رو داره که فقط با Ken Burns effect ساده کار می‌کنن (نه ویدیوی هوش‌مصنوعی واقعی). باید یک provider جدید برای Fal.ai/Seedance اضافه کنیم که واقعاً حرکت و صحنه بسازه، نه فقط زوم روی عکس ثابت.

## 📤 بخش ۳: انتشار خودکار (این مهم‌ترین کشفمه!)

### 🎉 خبر خوب: پروژه شما از قبل به بهترین گزینه وصله!

توی بررسی کدتون فهمیدم که `agents/publisher_zernio.py` از قبل به **Zernio** وصل شده — و طبق تحقیق بازار من، **Zernio دقیقاً ارزان‌ترین و منصفانه‌ترین گزینه بازاره**:

| سرویس | قیمت شروع | نکته |
|---|---|---|
| **✅ Zernio (از قبل در کدتون هست!)** | **رایگان تا ۲ اکانت** با پست نامحدود، بعد **$۶/اکانت/ماه** (۳ تا ۱۰ اکانت)، $۳ برای ۱۱-۱۰۰ | ۱۵ پلتفرم، شامل Instagram/TikTok/YouTube |
| Ayrshare | $149/ماه | خیلی گرون‌تر |
| Blotato | $29/ماه (flat) | شامل تولید عکس/ویدیو هم می‌شه ولی جدا از Zernio |
| Postiz (self-host) | رایگان (فقط هزینه سرور ~$5/ماه Railway) | باید خودتون میزبانی کنید |

**نتیجه: فقط باید در [zernio.com](https://zernio.com) ثبت‌نام کنید، اکانت‌های اینستاگرام/تیک‌تاک الینا رو وصل کنید، و کلید API (`sk_...`) رو در GitHub Secrets با نام `ZERNIO_API_KEY` بذارید.** بدون هیچ تغییر کدی، سیستم انتشار خودکار فعال می‌شه چون از قبل پیاده‌سازی شده و آزمایش شده (تست‌هاش پاس شدن)!

## 📋 جمع‌بندی: چه اکانت‌هایی باید بسازید

1. **[fal.ai](https://fal.ai)** — برای عکس (PuLID-FLUX) و ویدیو (Seedance) — نیاز به کارت (Payoneer/Wise هم جواب می‌ده)
2. **[zernio.com](https://zernio.com)** — برای انتشار خودکار — رایگان برای ۲ اکانت اول، بدون کارت
3. (اختیاری، اگه بودجه بیشتر دارید) فعال کردن billing در Google AI Studio برای برگردوندن Gemini

آیا می‌خواید با **Fal.ai** شروع کنیم؟ اگه کارت بانکی بین‌المللی (فیزیکی، Payoneer، یا Wise) دارید، می‌تونم قدم‌به‌قدم راهنماییتون کنم و بعد کد `image_studio.py` و `video_generator.py` رو برای اتصال واقعی به Fal.ai (به‌جای HuggingFace ZeroGPU محدود) بازنویسی کنم.
