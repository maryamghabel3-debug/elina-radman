## ✅ لیست کارهایی که باید انجام بدید (قدم به قدم)

### قدم ۱: ثبت‌نام xAI (برای عکس)
1. برید به **https://x.ai/api** یا **https://console.x.ai**
2. ثبت‌نام کنید (نیاز به شماره تلفن دارید، نه لزوماً کارت بانکی برای $۲۵ اعتبار اولیه)
3. یک API Key بسازید
4. در تنظیمات، گزینه «Data Sharing» یا مشابه رو فعال کنید تا $۱۵۰/ماه اعتبار اضافه بگیرید
5. کلید رو با نام `XAI_API_KEY` در GitHub Secrets ذخیره کنید

### قدم ۲: ثبت‌نام Fal.ai (برای ویدیو Seedance)
1. برید به **https://fal.ai**
2. ثبت‌نام کنید (این یکی معمولاً کارت بانکی می‌خواد، حتی برای اعتبار رایگان اولیه — اگه نداشتید بگید تا گزینه جایگزین پیدا کنم)
3. کلید API بسازید، با نام `FAL_KEY` در Secrets ذخیره کنید

### قدم ۳: Zernio (رایگان، برای انتشار — از قبل در کد پیاده‌سازی شده!)
1. **https://zernio.com** ثبت‌نام کنید
2. اکانت اینستاگرام/تیک‌تاک الینا رو وصل کنید (تا ۲ تا رایگانه)
3. کلید API (`sk_...`) رو با نام `ZERNIO_API_KEY` در Secrets بذارید

### قدم ۴ (کاری که من انجام می‌دم): به‌روزرسانی کد
بعد از این‌که اکانت‌ها ساخته شدن و کلیدها رو دادید، من:
1. یک provider جدید در `image_studio.py` برای **Grok Imagine** اضافه می‌کنم (image-to-image با عکس مرجع الینا)
2. یک provider جدید در `video_generator.py`/`faceless_studio.py` برای **Seedance 2.0 از Fal.ai** اضافه می‌کنم
3. ترتیب اولویت (fallback order) رو طوری تنظیم می‌کنم که این دو تا اول امتحان بشن (چون بهترین کیفیت/قیمت رو دارن)، و بقیه (Gemini، HuggingFace) فقط fallback باشن
