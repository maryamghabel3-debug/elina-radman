# Free LLM Setup Guide

This guide explains how to set up the 8 free LLM providers used in `LLMRouter`.

1. **Gemini (Google)**: Register at Google AI Studio for a free API key.
2. **Groq**: Register at console.groq.com. Incredible for fast, real-time responses.
3. **OpenRouter**: Sign up for free tier access to Claude and other models.
4. **GitHub Models**: Utilize the free tier available to GitHub users for GPT-4o/Llama-3.
5. **DeepSeek**: Register for free API credits. Best for coding tasks.
6. **Zhipu GLM-Flash**: Excellent Chinese/Global model, free tier available.
7. **Moonshot Kimi**: Great for long-context tasks.
8. **Poe**: Use as an aggregator for testing various models via API wrappers.

*Note: Always keep your API keys in a `.env` file and never commit them to the repository.*
