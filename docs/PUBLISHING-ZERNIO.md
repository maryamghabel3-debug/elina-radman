# 📤 راه‌اندازی انتشار خودکار با Zernio (رایگان)

این راهنما قدم‌به‌قدم می‌گه چطور انتشار خودکار الینا رو با **Zernio** (که قبلاً getlate.dev بود) راه بندازی. کاملاً از داخل مرورگر، بدون سرور، بدون App Review.

## 💰 هزینه
- **پلن رایگان:** ۲ اکانت متصل، پست نامحدود، API کامل، بدون کارت اعتباری.
- برای اکانت‌های بیشتر بعداً می‌تونی ارتقا بدی، یا به BrightBean self-host (نامحدود و رایگان) مهاجرت کنی.

---

## قدم ۱ — ثبت‌نام
1. برو به **https://zernio.com**
2. با ایمیل الینا **Sign up** کن.

## قدم ۲ — اتصال اکانت‌های الینا
1. توی داشبورد، یک **Profile** بساز (مثلاً «Elina»).
2. روی **Connect account** بزن و اکانت‌های الینا رو وصل کن (تا سقف ۲ اکانت در پلن رایگان — پیشنهاد: **Instagram + TikTok**).
3. هر اکانت با OAuth وصل می‌شه (فقط لاگین و تأیید).

> 💡 چون پلن رایگان ۲ اکانته، بهترین کار اینه که دو پلتفرم اصلی درآمدزا رو انتخاب کنی. بقیه رو بعداً با BrightBean اضافه می‌کنیم.

## قدم ۳ — گرفتن API Key
1. برو به **Settings → API Keys**
2. **Create API Key** رو بزن.
3. کلید با `sk_` شروع می‌شه — **همون لحظه کپیش کن** (دیگه نشون داده نمی‌شه).

## قدم ۴ — گذاشتن کلید در GitHub
1. برو به repo → **Settings → Secrets and variables → Actions → New repository secret**
2. Name: `ZERNIO_API_KEY`
3. Secret: کلیدی که کپی کردی → **Add secret**

## قدم ۵ — تست
- **از تلگرام:** یک محتوا بساز (`/content`)، تأییدش کن (`/approve شناسه`)، بعد `/publish` بزن.
- **یا** توی repo: **Actions → Publish Content → Run workflow**.

اگه درست باشه، پست در اکانت‌های وصل‌شده منتشر/زمان‌بندی می‌شه. ✅

---

## چطور کار می‌کنه؟
سیستم به‌صورت خودکار backend انتخاب می‌کنه:
- اگه `ZERNIO_API_KEY` ست باشه → **Zernio**
- وگرنه اگه `POSTIZ_API_TOKEN` باشه → **Postiz**

پلتفرم‌های هر پست از فیلد `platforms` در صف محتوا خونده می‌شن و فقط اون‌هایی که در Zernio وصل شدن ارسال می‌شن (بقیه بی‌صدا رد می‌شن).

## عیب‌یابی
| پیام | معنی | راه‌حل |
|------|------|--------|
| `no_token` | کلید ست نشده | `ZERNIO_API_KEY` رو در Secrets بذار |
| `no_accounts` | هیچ اکانتی وصل نیست | در داشبورد Zernio اکانت وصل کن |
| `HTTP 4xx` رد شد | مشکل محتوا/دسترسی | متن/پلتفرم رو چک کن؛ اکانت هنوز وصله؟ |

## مهاجرت به نامحدود (بعداً)
وقتی خواستی همه‌ی اکانت‌ها (اینستا + تیک‌تاک + یوتیوب + پینترست + فیسبوک + ...) رایگان باشن، به **BrightBean Studio** (متن‌باز، self-host) مهاجرت می‌کنیم. کد الینا از قبل چند-backend رو پشتیبانی می‌کنه، پس اضافه کردنش ساده‌ست.
