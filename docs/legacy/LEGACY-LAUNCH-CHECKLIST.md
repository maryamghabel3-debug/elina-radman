# 🚀 Elina Radman - Launch Checklist & Testing Guide

این فایل لیست کارهای عملی (دنیای واقعی) است که باید برای اجرای نهایی پروژه الینا انجام شود، به همراه راهنمای تست خروجی‌ها.

## 🛠️ بخش اول: ثبت‌نام‌ها و دریافت API (کارهای انسانی)

برای اینکه ایجنت‌های ما بتوانند کارشان را واقعی انجام دهند، شما باید در سایت‌های زیر ثبت‌نام کنید و کلیدها (API/Tokens) را بگیرید:

### ۱. پلتفرم‌های انتشار و هوش مصنوعی
- [ ] **Cloudflare Workers AI:** ثبت‌نام رایگان در Cloudflare برای ساخت بی‌نهایت عکس B-Roll. (نیاز به `CF_ACCOUNT_ID` و `CF_API_TOKEN`).
- [ ] **Google AI Studio (Gemini):** دریافت API Key رایگان برای موتور تولید متن (`ContentCreator`).
- [ ] **Postiz (یا Buffer):** ثبت‌نام برای اتصال به شبکه‌های اجتماعی الینا جهت انتشار خودکار محتوا.
- [ ] **Hugging Face:** ساخت یک اکانت برای اجرای مدل‌های ساخت ویدیو (VideoGenerator) در فضای Spaces (رایگان).

### ۲. پلتفرم‌های درآمدزایی (Affiliate)
- [ ] **Linktree:** ساخت یک صفحه برای قرار دادن تمام لینک‌ها در Bio اینستاگرام و تیک‌تاک.
- [ ] **Amazon Associates:** ثبت‌نام در سیستم همکاری در فروش آمازون برای دریافت لینک‌های پوشاک و تجهیزات باغبانی.
- [ ] **LTK (LikeToKnowIt) / RewardStyle:** ثبت‌نام به عنوان اینفلوئنسر فشن (بهترین برای برندهایی مثل Aritzia و Reformation).
- [ ] **Gumroad:** ساخت اکانت برای فروش فایل‌های دیجیتال در آینده (مثل پریست عکس).

## 🧪 بخش دوم: تست عملی سیستم (Quality Assurance)

قبل از عمومی کردن پیج الینا، باید کیفیت عکس‌ها و ویدیوها را تست کنیم:

### تست ۱: ثبات چهره (Consistent Face Test)
- **ابزار:** ComfyUI محلی (اگر سیستم قوی دارید) یا Replicate / Fal.ai (برای سرویس ابری).
- **مدل پیشنهادی:** `InstantID` یا `FaceID` به همراه یک مدل `SDXL`.
- **نحوه تست:** 
  ۱. ۵ عکس واضح از دختری که چهره‌اش به توصیفات الینا (خاورمیانه‌ای، مینیمال، موهای خرمایی) نزدیک است بسازید یا پیدا کنید.
  ۲. این ۵ عکس را به عنوان Reference (مرجع) به ابزار بدهید.
  ۳. از سیستم بخواهید ۳ عکس مختلف بسازد (الینا در کافه، الینا در گلخانه، الینا در حال قدم زدن در پاریس).
  ۴. **معیار قبولی:** چهره در هر ۳ عکس باید دقیقاً یک نفر باشد.

### تست ۲: کیفیت صدای الینا (TTS Test)
- **نحوه تست:** ایجنت `Director` را با دستور `edge-tts --voice fa-IR-DilaraNeural --text "سلام دخترا🤍 امروز می‌خوایم درباره روانشناسی استایل صحبت کنیم."` تست کنید.
- **معیار قبولی:** صدا باید گرم، طبیعی و بدون قطعیِ رباتیک باشد.

### تست ۳: خروجیِ ویدیو (Video Test)
- **نحوه تست:** یک ویدیوی ۱۰ ثانیه‌ای بدون حضور الینا (B-Roll از یک کافه یا باغ) توسط Cloudflare/HuggingFace بسازید.
- صدای تست شده در مرحله قبل را روی آن قرار دهید و زیرنویس اضافه کنید.
- **معیار قبولی:** هماهنگی صدا و متن، و وایرال‌پذیریِ ویدیو در قالب 9:16 (سایز Reels).

## 📌 نکته مهم اجرایی
اگر قصد استفاده از مدل‌های ابری پولی (مثل Replicate یا Fal.ai) برای "ثبات چهره" را دارید، باید در آن‌ها ثبت‌نام کرده و حسابتان را چند دلار شارژ کنید، زیرا ساخت چهره ۱۰۰٪ ثابت در حال حاضر به صورت کاملاً رایگان (بدون سیستم لوکال قوی) کیفیت بسیار پایینی دارد.
