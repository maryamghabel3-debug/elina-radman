# 🔎 بررسی ۲۲ ابزار پیشنهادی شما (۶ ژوئیه ۲۰۲۶)

همه‌ی این‌ها رو با جستجوی زنده بررسی کردم — نه از حافظه. برای هرکدوم: واقعاً رایگانه یا نه، و به پروژه‌ی الینا می‌خوره یا نه.

---

## ✅ دسته‌ی ۱: واقعاً رایگان و مستقیماً به کار الینا می‌خوره

### 1. **freellmapi** (`tashfeenahmed/freellmapi`) — ⭐ بهترین کشف این لیست
- **چیه:** یک پروکسی متن‌باز (MIT) که ۱۶+ ارائه‌دهنده‌ی رایگان LLM (Groq، Gemini، Cerebras، Mistral، OpenRouter، GitHub Models و...) رو پشت یک endpoint واحد و سازگار با OpenAI جمع می‌کنه. رویش داکر بالا میاد، داشبورد کامل داره، وقتی یکی quota تموم کرد خودکار می‌ره سراغ بعدی.
- **چرا به الینا می‌خوره:** `agents/llm_router.py` الینا همین الان یه نسخه‌ی ساده‌ی همین ایده رو دستی پیاده کرده (چند تا provider، انتخاب بر اساس کلید موجود). این پروژه یه نسخه‌ی خیلی پخته‌تر و آماده‌ست — می‌تونیم `LLMRouter` رو بازنویسی کنیم که به‌جای منطق دستی، از این پروکسی (که خودش روی سرور جدا یا حتی همون GitHub Actions به صورت سبک اجرا می‌شه) استفاده کنه و دیگه نیازی به مدیریت دستی fallback بین provider ها نباشه.
- **هزینه:** کاملاً رایگان و متن‌باز، فقط باید خودتون میزبانیش کنید (یک سرور کوچیک یا حتی لوکال).

### 2. **AgentRouter** (`agent-router.org`)
- **چیه:** یک gateway هوش مصنوعی که تماس ایجنت‌ها و MCP tool رو کاملاً رایگان می‌کنه (نه فقط credit اولیه — واقعاً همیشگی رایگان برای استفاده‌ی عادی).
- **چرا به الینا می‌خوره:** به‌عنوان یه provider اضافی در `llm_router.py` قابل اضافه‌شدنه، مخصوصاً چون محدودیت نرخ نداره (فقط برای IP ناشناس محدودتره، با ثبت‌نام رایگان محدودیتش بازتر می‌شه).

### 3. **HALO** (`context-labs/HALO`)
- **چیه:** یک ابزار متن‌باز که ردپای اجرای ایجنت‌ها (trace) رو تحلیل می‌کنه، الگوهای شکست رو پیدا می‌کنه، و گزارشی می‌ده که به یک ایجنت کدنویس (مثل Claude Code) بدید تا خودکار کد رو اصلاح کنه.
- **چرا به الینا می‌خوره:** می‌تونه برای دیباگ کردن ایجنت‌های `DirectorAgent`/`ContentCreator` وقتی رفتار غیرمنتظره دارن مفید باشه — ابزار "چرا ایجنت این تصمیم رو گرفت" دقیقاً چیزیه که با بزرگ‌تر شدن تعداد ایجنت‌های الینا لازم می‌شه.

### 4. **caveman** (`JuliusBrussee/caveman`)
- **چیه:** یک skill برای فشرده‌سازی خروجی مکالمات ایجنت‌های کدنویس (کاهش ۶۵٪ توکن مصرفی) بدون از دست دادن دقت فنی.
- **چرا به الینا می‌خوره:** غیرمستقیم مفیده — اگه از Claude Code/ابزارهای مشابه برای توسعه‌ی الینا استفاده می‌کنید، هزینه‌ی توکن مصرفی رو کم می‌کنه. ربطی به خود کد الینا نداره، فقط ابزار کمکی توسعه‌ست.

## ⚠️ دسته‌ی ۲: رایگان محدود (نه کاملاً رایگان) — مفید ولی با احتیاط

| ابزار | نوع رایگان بودن | نکته |
|---|---|---|
| **Google Flow** (`flow.google`) | رایگان با محدودیت روزانه (اعتبار محدود، کیفیت پایین‌تر Veo Lite) | برای تولید تصویر/ویدیوی با کیفیت بالا نیاز به پرداخت داره؛ برای تست اولیه مفیده |
| **HeyGen** | ۳ ویدیو در ماه، هرکدوم حداکثر ۳ دقیقه، رایگان | دقیقاً کاری که ما با `LipSyncStudio` سعی کردیم رایگان بسازیم رو انجام می‌ده، ولی محدودیت ماهانه داره. برای دوبله‌ی ویدیوهای الینا به چند زبان (فارسی/انگلیسی) با محدودیت ماهانه قابل استفاده‌ست |
| **Meigen.ai** | مرور و کپی پرامپت رایگان؛ اجرا (Run) نیاز به ورود با گوگل داره و مصرف اعتبار داره | فقط برای الهام گرفتن از پرامپت‌های آماده مفیده، نه اجرای واقعی رایگان نامحدود |
| **ZenMux.ai** | اعتبار آزمایشی محدود برای Claude؛ بعدش قیمت معمولی Anthropic | برای تست مدل Claude خوبه، نه یک منبع همیشگی رایگان |
| **conduit.ozdoev.net** | ۵۰۰ دلار اعتبار آزمایشی (نه همیشگی)، نیاز به عضویت کانال تلگرام | یه گیت‌وی API با اعتبار آزمایشی، بعد از تمومش پولیه |
| **atxp.chat** | ۳ دلار اعتبار اولیه رایگان، بعدش pay-per-message | برای گفتگوی محدود با چند مدل، نه استفاده‌ی تجاری مداوم |
| **Zyloo.io / zyloai.net** | پلن پایه رایگان با محدودیت روزانه و نرخ درخواست (۱۰ در دقیقه) فقط مدل‌های ساده | برای مصرف سبک (نه اصلی) قابل استفاده‌ست |
| **Freemodel.dev** | اعتبار اولیه رایگان (حدود ۱۰-۲۰ دلار)، بعد محدودیت هفتگی | شبیه ZenMux، برای شروع مفید ولی محدود |
| **Gumloop.com** | ۲۰۰۰ اعتبار رایگان (نه ۵۰۰۰ - این عدد رو در تحقیق زنده تأیید نکردم، رقم واقعی که پیدا کردم ۲۰۰۰ بود) | ابزار اتوماسیون بصری (شبیه n8n)؛ برای پروژه‌های کوچک تست خوبه |

## ❌ دسته‌ی ۳: مربوط نیست یا مشکل‌دار

- **Okara.ai:** یک پلتفرم مدیریت‌شده (نه متن‌باز) برای بازاریابی محتوا (SEO/Reddit/GEO) با پلن رایگان بسیار محدود (۵ اعتبار). این برای بازاریابی کسب‌وکارهای عادی طراحی شده، نه برای اینفلوئنسر شخصی مثل الینا — تطابق مستقیم نداره.
- **Suno.com:** برای تولید آهنگ کامل (نه فقط موزیک پس‌زمینه) است — رایگانش محدود به چند آهنگ در ماهه. برای پروژه‌ی الینا (که نیاز به موزیک پس‌زمینه‌ی ساده داره، نه آهنگ کامل) زیاده‌روی محسوب می‌شه؛ برای یوتوب فکتوری هم `music_library.py` (Kevin MacLeod رایگان و بدون محدودیت) از قبل این نیاز رو حل کرده.
- **HunyuanVideo.org:** این یه دامنه‌ی شخص ثالثه، نه سایت رسمی Tencent — از قبل در `agents/video_generator.py` از طریق Hugging Face Space رسمی (`tencent/HunyuanVideo`) استفاده می‌کنیم که امن‌تره.
- **Google Omni (Gemini Omni):** بخشی از Gemini API معمولیه (نه ابزار جدا)، از قبل با `GEMINI_API_KEY` قابل دسترسیه — چیز جدیدی برای اضافه کردن نیست.
- **HelixDB:** یک دیتابیس گراف/برداری برای RAG (جستجوی معنایی) است. الینا فعلاً هیچ نیاز واقعی به این سطح از دیتابیس نداره (حافظه‌ی فعلی `memory_engine.py` با فایل JSON ساده کافیه) — اضافه کردنش پیچیدگی بی‌مورد میاره.
- **affaan-m/ECC:** یک پلاگین برای Claude Code (نه برای الینا مستقیم) — مربوط به تجربه‌ی توسعه‌دهنده، نه کد پروژه.
- **21st.dev / motion.dev / Higgsfield MCP:** همگی ابزارهای طراحی وب/UI برای React هستن — فقط اگه بخواید یک وب‌سایت/داشبورد جدید برای الینا بسازید به کار میان، نه به پایپ‌لاین محتوای فعلی.

### 🚨 هشدار امنیتی/اخلاقی — یک مورد
- **ShadowHackers/gmail-account-creator:** این یک ابزار برای ساخت انبوه و خودکار حساب Gmail با «دور زدن تشخیص ربات» و «دور زدن تأیید شماره تلفن» است. **این دقیقاً همون نوع فعالیتی است که سیاست‌های Google را نقض می‌کند و می‌تواند به مسدود شدن IP/دامنه یا حتی مسائل حقوقی منجر شود.** به‌شدت پیشنهاد می‌کنم از این ابزار **استفاده نشود** — نه برای الینا و نه برای یوتوب فکتوری. اگر نیاز به چند حساب گوگل واقعی برای کانال‌های مختلف دارید، بهتره دستی و طبق قوانین گوگل ثبت بشن.
