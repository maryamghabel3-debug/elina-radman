## 📝 PER-PLATFORM LANGUAGE STRATEGY

### TikTok
- **Separate content strategy** per language
- English: Full scripts, voiceover, captions in English
- Persian: Same concept but re-filmed in Persian (or voiceover)
- Use trending sounds from each region (different audios!)

### Instagram
- **Bilingual captions** on every post
- First line: English (hooks global scrollers)
- Below: Persian translation
- Stories: Mix English and Persian polls/stickers

### YouTube
- **English primary**, Persian subtitles
- Long-form videos in English (monetization is higher)
- Add high-quality Persian subtitles (not auto-generated!)
- Separate "Persian Only" playlist with re-dubbed content

### Pinterest
- **English only** — it's a search engine, English dominates
- Pin titles and descriptions in English

Why different? Persian audience has different needs:

- Hijab-friendly Quiet Luxury styling

## 🔄 CONTENT LOCALIZATION WORKFLOW

```
1. CREATE → Film in English (primary)
2. CAPTION → Add English auto-captions
3. TRANSLATE → Write Persian script/caption
4. PUBLISH → Post English version globally
5. LOCALIZE → Re-dub or re-film for Persian/Arabic
6. PUBLISH → Post localized version 1-2 days later
```
