# 🤖 راهنمای کامل دستورات ربات تلگرام الینا

هر دستوری که در تلگرام به ربات می‌فرستی، این کار رو انجام می‌ده:

## 📊 مدیریت و وضعیت

| دستور | چیکار می‌کنه |
|-------|--------------|
| `/start` | پیام خوش‌آمدگویی + منوی کوتاه دستورها |
| `/status` | وضعیت سیستم: تعداد فایل‌های صف، ایجنت‌ها، و اتصالات (Gemini / Postiz / GitHub ✅❌) |
| `/help` | لیست کامل همه‌ی دستورها |
| `/agents` | لیست پویا‌ی ایجنت‌های موجود در پوشه `agents/` |
| `/github` | اطلاعات مخزن: تعداد workflowها و ایجنت‌ها (پویا) |

## 🎨 ساخت و مدیریت محتوا

| دستور | چیکار می‌کنه |
|-------|--------------|
| `/content` | **۳ پست جدید می‌سازه** با ContentCreator (کپشن با Gemini + هشتگ + بر اساس ترند زنده). در صف `pending_approval` ذخیره می‌شه |
| `/list` | ۵ فایل آخر صف محتوا رو با وضعیتشون نشون می‌ده (⏳ در انتظار / ✅ تأییدشده / ❌ ردشده) |
| `/approve شناسه` | یک پست رو **تأیید** می‌کنه (وضعیتش `approved` می‌شه تا منتشر بشه). مثال: `/approve elina-20260701-ootd` |
| `/reject شناسه` | راهنمای رد کردن یک پست |
| `/publish` | پست‌های **تأییدشده** رو منتشر می‌کنه (از Zernio اگه ست باشه، وگرنه Postiz) |

## 🔥 ترند و تحلیل

| دستور | چیکار می‌کنه |
|-------|--------------|
| `/trends` | ترندهای واقعی فشن رو نشون می‌ده (از Reddit + YouTube + Google Trends) |
| `/photo [توضیح]` | **یک عکس واقعی از الینا می‌سازه** (Gemini → Pollinations). مثال: `/photo کت شتری در کافه پاریسی` — اگه توضیح ندی، یک لوک پیش‌فرض می‌سازه |
| `/topimages` | عکس‌هایی که بیشترین ویو/محبوبیت رو دارن + لینکشون |
| `/analyze` | **تحلیل عمیق عکس‌های ترند**: استایل، محصولات شاخص، زاویه دوربین، پالت رنگ (با Gemini Vision) |
| `/reverse` | **مهندسی معکوس ویدیوهای وایرال**: موضوع، هوک، ساختار، و دلیل وایرال شدن |
| `/makevideos` | از تحلیل ویدیوها، **ایده‌های ویدیو آماده** برای الینا می‌سازه (نیاز به اجرای `/reverse` قبلش) |

## 📓 دفترچه احساسات

| دستور | چیکار می‌کنه |
|-------|--------------|
| `/diary متن` | احساس امروز الینا رو ثبت می‌کنه؛ روی لحن کپشن‌های تولیدی تأثیر می‌ذاره. مثال: `/diary امروز پرانرژی و الهام‌گرفته‌ام` |

## 💬 چت هوشمند

هر پیامی که **دستور نباشه** (با `/` شروع نشه) → مستقیم با **Gemini** به‌عنوان شخصیت الینا چت می‌شه.
مثال: «امروز برای یک قرار کاری چه استایلی پیشنهاد می‌دی؟»

---

## 🔄 گردش کار معمول (workflow روزانه‌ی تو)

```
۱. /content            → ۳ پست جدید ساخته می‌شه
۲. /list               → پست‌ها و شناسه‌هاشون رو ببین
۳. /approve <شناسه>    → پستی که دوست داری رو تأیید کن
۴. /publish            → پست‌های تأییدشده منتشر می‌شن
```

یا برای ویدیو:
```
۱. /reverse       → ویدیوهای وایرال تحلیل می‌شن
۲. /makevideos    → ایده‌های ویدیو ساخته می‌شن
۳. /list → /approve → /publish
```

> 💡 نکته: خط تولید **خودکار روزانه** هم هست که هر شب بدون دخالت تو محتوا می‌سازه و در تلگرام برات می‌فرسته. این دستورها برای وقتیه که خودت دستی بخوای کاری کنی.
